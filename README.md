# document

## README: Документ о выплатах

**Описание программы**
Консольное приложение для генерации уведомления о выплатах работнику.

**Функционал**
- Считывает дату начала работ, ФИО и три суммы выплат
- Вычисляет дату подписания договора (дата начала + 15 дней)
- Подсчитывает общую сумму выплат (рубли и копейки)
- Формирует официальное уведомление по заданному шаблону

**Технологии**
- Язык: Go
- Пакеты: fmt, time, math

**Формат ввода**
- Дата: `dd.MM.yyyy`
- ФИО: три строки
- Суммы: три числа float64

**Формат вывода**
Готовое уведомление с подписью главного бухгалтера
